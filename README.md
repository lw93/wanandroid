# wanandroid
本项目基于 http://www.wanandroid.com/ 进行 MVP + OKHttp + RxJava + Retrofit + Glide + EventBus 等架构设计，努力打造一款优秀开源的可共同学习的App。感谢各位朋友的开源项目！
